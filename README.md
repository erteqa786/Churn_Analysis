# 📊 Customer Churn Analysis

## 🎯 Project Objective

To analyze customer data and identify key factors driving churn, enabling the business to improve customer retention and reduce revenue loss.

---

## 🧠 Problem Statement

A telecom company is facing a high churn rate, leading to:

* Loss of revenue
* Increased customer acquisition costs
* Poor customer experience

The goal is to understand:

* Why customers are leaving
* Which customers are likely to churn
* How churn can be reduced

---

## 📂 Dataset Description

The dataset contains 1000 customer records with the following features:

* **Demographics:** Age, Gender
* **Account Info:** Tenure, Contract Type
* **Financial:** Monthly Charges, Total Charges
* **Services:** Internet Service, Tech Support, Online Security
* **Behavioral:** Complaints, Satisfaction Score
* **Billing:** Payment Method, Paperless Billing
* **Target:** Churn (Yes/No)

---

## 🛠️ Tools & Technologies

* **Excel** → Initial data exploration
* **PostgreSQL** → Data querying and KPI calculation
* **Python** → Data cleaning, EDA, and modeling
* **Power BI** → Dashboard and visualization

---

## 🔄 Project Workflow

1. **Data Cleaning**

   * Handled missing values
   * Corrected data types
   * Created new features (Age Group, Flags)

2. **Exploratory Data Analysis**

   * Churn distribution analysis
   * Segment-based insights
   * Correlation analysis

3. **SQL Analysis**

   * Churn rate calculation
   * Revenue loss estimation
   * Customer segmentation
   * Multi-factor analysis

4. **Machine Learning**

   * Logistic Regression model
   * Identified key churn drivers
   * Model evaluation (accuracy)

5. **Power BI Dashboard**

   * KPI tracking
   * Interactive segmentation
   * Business insights visualization

---

## 📊 Key Insights

* 📄 Customers on **Month-to-Month contracts** have the highest churn
* 😡 Higher number of **complaints significantly increases churn**
* 😊 Lower **satisfaction scores strongly correlate with churn**
* 👥 **Senior and Young Adult** segments show higher churn rates
* 📞 Lack of **tech support** increases churn probability

---

## 🤖 Model Insights

* Built a **Logistic Regression model** to predict churn
* Key drivers:

  * Complaints → Increase churn
  * Satisfaction Score → Decreases churn
* Model helps identify **high-risk customers**

---

## 💰 Business Impact

* Identified key drivers affecting customer churn
* Estimated revenue loss due to churn
* Enabled targeted retention strategies

---

## 🚀 Business Recommendations

* Offer incentives for **long-term contracts**
* Improve **customer support systems**
* Reduce complaint resolution time
* Target **high-risk customer segments**
* Enhance overall customer satisfaction
